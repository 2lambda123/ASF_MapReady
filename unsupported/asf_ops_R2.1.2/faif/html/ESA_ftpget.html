  1. Description of the ESA_ftpget.csh and ESA_ftpget_all.csh (C shell) scripts.
  2. Summary argument list for the scripts
  3. Valid argument values for the scripts
  4. Environment variables to set
  5. Other programs used, therefore required, by these scripts
  1. SCRIPT NAMES: ESA_ftpget.csh and ESA_ftpget_all.csh

ESA_ftpget.csh is the FTP script used to pull files from the ESA host machine to the FAIF host machine. Note that this script is normally activated only from ESA_ftpget_all.csh. Execution on the command line is not usually performed.

In normal operations, ESA_ftpget_all.csh is the job that appears in the FAIF crontab file. ESA_ftpget_all.csh first uses set_env.csh to establish necessary environment variables (see (4) below), then invokes ESA_ftpget.csh repeatedly, once for each incoming ERS-1 and ERS-2 file type being retrieved.

After ERS-1 and ERS-2 files are retrieved, ESA_ftpget_all.csh invokes ESAdirmon to process them.

2. Arguments required

ESA_ftpget_all.csh takes no arguments.

Arguments required when executing ESA_ftpget.csh: 

ARGUMENT 1:  RemoteHost               ex. ersiss.esrin.esa.it
ARGUMENT 2:  upw file                 ex. $FAIF_ROOTPATH/FTP/upw/ESAsend.upw
ARGUMENT 2:  SourceDir                ex. from_eecf
ARGUMENT 3:  Receptdir                ex. $FAIF_ROOTPATH/ESA
ARGUMENT 5:  FileToGetType            ex. PATC
ARGUMENT 6:  Mode                     ex. binary

3. Valid argument 5 values for ESA_ftpget.csh 

PATC
ORPD
ORRE
ODMC
ODMR
SHAQ
RQST
RQVR
MPSG

4. Environment variables

As explained shortly, setting necessary environment variables is automatic in normal operations.

The FAIF_BINPATH environment variable must be set prior to running this script or running a FAIF program that invokes this script. $FAIF_BINPATH is normally "/asf/bin" in the operational environment.

The FAIF_ROOTPATH environment variable must also be set prior to running a FAIF program that invokes this script. This is normally "/local/faif" in the operational environment.

Both of these environment variables are established in set_env.csh. In normal operations, ESA_ftpget_all.csh, which is in the crontab file to retrieve all ERS-1 and ERS-2 files from ESA, first uses set_env.csh, then repeatedly invokes ESA_ftpget.csh.

5. Other files needed to run successfully

$HOME/set_env.csh
$FAIF_ROOTPATH/FTP/upw/ESAsend.upw
