  1. SCRIPT NAME: CSA_ftpget.csh

This is the FTP script used to pull files from the CSA host machine to the FAIF host machine. Note that this script is activated within FAIF programs only. Execution on the command line is not usually performed.

2. Arguments required when executing CSA_ftpget.csh 

ARGUMENT 1:  RemoteHost               ex. mcs_vdn
ARGUMENT 2:  RemoteDir / SourceDir    ex. mcs_to_ufdroc_1
ARGUMENT 3:  Receptdir                ex. /local/faif/CSA/CSA_PREDORBIT
ARGUMENT 4:  FileToGetType            ex. CSA_PREDORBIT
ARGUMENT 5:  Mode                     ex. ascii
ARGUMENT 6:  FAIF_ROOTPATH            ex. /local/faif
ARGUMENT 7:  Username                 ex. pps 
ARGUMENT 8:  Password                 ex. Jkd*%s

3. Valid argument 4 values 

CSA_PREDORBIT
CSA_DEFVORBIT
CSA_RECRQST
CSA_RECSCHED
CSA_CALIBRQST
CSA_CALIBSCHED
CSA_SARPROCPRM
CSA_RRQ_MCM
CSA_RSH_MCM
CSA_CRQ_MCM
CSA_CSH_MCM

4. Environment variables to set

The FAIF_BINPATH environment variable must be set prior to running this script or running a FAIF program that invokes this script. $FAIF_BINPATH is normally "/asf/bin" in the operational environment.

The FAIF_ROOTPATH environment variable must also be set prior to running a FAIF program that invokes this script. This is normally "/local/faif" in the operational environment.

5. Other programs which CSA_ftpget.csh needs to run successfully

$FAIF_BINPATH/idfilename
$FAIF_BINPATH/CSAPredOrRes

6. Files created by this script

Note that the environment variable $FAIF_ROOTPATH is normally defined before running the FAIF program that invokes this script. It is then passed as ARGUMENT 6 to this script.

$FAIF_ROOTPATH/FTP/gtmp/remotelisting.$FileToGetType $FAIF_ROOTPATH/FTP/gtmp/remotelisting2.$FileToGetType $FAIF_ROOTPATH/FTP/gtmp/CSA_ftpget.activity.$FileToGetType $FAIF_ROOTPATH/FTP/gtmp/tmp1.tmp1.$FileToGetType $FAIF_ROOTPATH/FTP/gtmp/tmp2.tmp2.$FileToGetType $FAIF_ROOTPATH/FTP/gtmp/ftypedlistfile.$FileToGetType

7. Other notes

Note that orbit files obtained are stored in the following directory while their precision (predicted or restituted) is being determined:

$FAIF_ROOTPATH/CSA/CSA_ORBIT
