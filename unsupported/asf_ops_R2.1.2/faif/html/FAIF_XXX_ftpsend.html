  1. SCRIPT NAME: FAIF_XXX_ftpsend.csh (FAIF_VAX_ftpsend.csh for VAX FTP, FAIF_UNIX_ftpsend for Unix FTP)

This is the FTP script used to send files from the FAIF host machine to the external (Vax or Unix) host machines. Note that this script is activated within FAIF programs only. Execution on the command line is not usually performed.

2. Arguments required when executing CSA_ftpget.csh 

ARGUMENT 1:  RemoteHost     ex. mcs_vdn
ARGUMENT 2:  RemoteDir      ex. mcs_to_ufdroc_1
ARGUMENT 3:  FileToSend     ex. m0123456.rsh
ARGUMENT 4:  Mode           ex. ascii
ARGUMENT 5:  Rootpath       ex. /local/faif
ARGUMENT 6:  Username       ex. pps 
ARGUMENT 7:  Password       ex. Jkd*%s

3. Files created by this script

Note $Rootpath is from ARGUMENT 5 above. 

$Rootpath/FTP/stmp/sentcopy/$FileToSend
$Rootpath/FTP/stmp/$FileToSend.tmp
$Rootpath/FTP/stmp/diff.result.$FileToSend
